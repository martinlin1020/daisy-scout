# Daisy Scout v0.1

Semi-automatic product opportunity screener for Daisy's Breeze.

## Deploy to Streamlit Cloud
1. Create a GitHub repository.
2. Upload these files.
3. Go to Streamlit Community Cloud.
4. Deploy `app.py`.

## Local run, optional
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Important
This version does not scrape Amazon or Shopee.
